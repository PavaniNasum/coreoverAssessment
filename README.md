# Chatbot Dashboard (Angular 18)

# Steps to Run the Project

1. Clone the repository

```
git clone https://github.com/PavaniNasum/coreoverAssessment
```

2. Install dependencies

```
npm install
```

3. Install required packages

```
npm install bootstrap bootstrap-icons chart.js ng2-charts@4
```

4. Add Bootstrap in angular.json

```
"styles": [
  "node_modules/bootstrap/dist/css/bootstrap.min.css",
  "node_modules/bootstrap-icons/font/bootstrap-icons.css",
  "src/styles.scss"
]
```

5. Run the project

```
ng serve
```

6. Open in browser

```
http://localhost:4200
```

* Chatbot with static responses
* Dashboard with charts
